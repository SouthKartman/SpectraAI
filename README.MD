## 📖 О проекте

**Spectrum** – это полностью голосовой ассистент с визуальным откликом в виде анимированных частиц.  
Он работает в паре с сервером на FastAPI + Groq (LLM) и умеет:

- 🔹 Распознавать речь (Google Speech)  
- 🔹 Отвечать голосом (edge‑tts + потоковое воспроизведение)  
- 🔹 Совершать и принимать **звонки** (WebSocket + аудио‑ретрансляция)  
- 🔹 Отправлять и читать **сообщения** (REST + WebSocket)  
- 🔹 Управлять **контактами** (добавление, удаление, поиск по падежам)  
- 🔹 Ставить **таймеры** и **напоминания**  
- 🔹 Менять **цвет интерфейса**, **скорость/громкость** речи, **режим экономии**  
- 🔹 Открывать **YouTube** по команде  
- 🔹 Поддерживать **диалог** через нейросеть (сложные вопросы обрабатывает большая модель, простые – быстрая)

Интерфейс – окно 600×600 с **частицами**, которые меняют форму в зависимости от режима (телефон, конверт, контакты, мышление, речь).

### 📞 Звонки

| Команда / Пример | Действие |
|----------------|----------|
| `Спектр, позвони Диме` (или любая фраза из `CALL_PHRASES` + имя) | Находит контакт (с учётом падежей) и инициирует звонок. |
| `Спектр, позвони` (без имени) | Включает режим звонка и спрашивает «Кому позвонить?». |
| `положить трубку`, `отбой`, `завершить звонок` (во время разговора) | Завершает активный звонок. |
| **Входящий звонок:** ассистент спрашивает «Ответить?» → вы говорите:<br/>`да` / `подтверждаю` / `принимаю` / `ок` / `давай` / `конечно` / `алло` / `принято` | Принимает звонок. |
| → вы говорите:<br/>`нет` / `отклоняю` / `не надо` / `отбой` / `откажись` / `позже` / `не сейчас` / `отказать` | Отклоняет звонок. |

### ✉️ Сообщения

| Команда / Пример | Действие |
|----------------|----------|
| `Спектр, напиши Диме привет` (любая фраза из `MESSAGE_PHRASES` + имя + текст) | Отправляет сообщение. |
| `Спектр, напиши Диме` (имя есть, текста нет) | Спрашивает «Что отправить?», затем подтверждение. |
| `Спектр, напиши` (без имени) | Включает режим конверта, спрашивает получателя, потом текст. |
| `прочитай сообщения`, `покажи сообщения`, `новые сообщения` | Зачитывает непрочитанные сообщения, предлагает прочитать. |
| (после прочтения) `да` / `ответить` | Переходит к отправке ответа. |
| (в диалоге отправки) `да` / `подтверждаю` | Отправляет сообщение. |
| (в диалоге отправки) `нет` / `отмена` | Отменяет отправку. |

### 🧠 Общие вопросы (отправляются на сервер Groq)

Любая фраза, не подпадающая под команды выше, отправляется на сервер и обрабатывается нейросетью.  
Примеры:  
- «Спектр, какая сегодня погода?»  
- «Спектр, расскажи анекдот»  
- «Спектр, сколько будет 2+2?»

## 🔁 Многошаговые диалоги

| Состояние | Ожидаемый ответ | Действие |
|-----------|----------------|----------|
| `waiting_for_contact_name` | Имя контакта | Запрос ID |
| `waiting_for_contact_id` | Цифры ID (словами или цифрами) | Подтверждение |
| `waiting_for_contact_confirm` | `да` / `подтверждаю` или `нет` | Добавление / отмена |
| `waiting_for_contact_delete` | Имя контакта | Удаление |
| `waiting_for_message_contact` | Имя получателя | Запрос текста |
| `waiting_for_message_text` | Текст сообщения | Подтверждение отправки |
| `waiting_for_message_confirm` | `да` / `подтверждаю` или `нет` | Отправка / отмена |
| `waiting_for_message_read` | `да` / `прочитай` или `нет` / `позже` | Чтение / пропуск |
| `waiting_for_message_reply` | `да` / `ответить` или `нет` / `не надо` | Ответ / отказ |
| `waiting_for_call_name` | Имя контакта для звонка | Начало звонка |
| `waiting_for_call_answer` | `да` / `подтверждаю` или `нет` / `отклоняю` | Принять / отклонить |
